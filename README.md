## dependencies

> nodejs [get latest here](https://nodejs.org/en/)
>
> yarn [get latest here](https://yarnpkg.com/getting-started/install)

## dev

```bash
git clone https://github.com/bamdadsabbagh/screwmycode-ext.git

cd screwmycode-ext

yarn install

# in 2 tabs
yarn dev
yarn start
```

## build

```bash
yarn build
```
